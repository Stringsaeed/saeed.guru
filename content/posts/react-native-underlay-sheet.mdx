I stumbled on this animation a couple of weeks ago on Twitter; it was a beautiful prototype from [@JordanAmblin](https://twitter.com/JordanAmblin).

<Tweet id="1765047420390445293" />

So I decided to recreate it in React Native using react-native-reanimated, react-native-gesture-handler, and [@gorhom/bottom-sheet](https://github.com/gorhom/react-native-bottom-sheet).

The goal of this demo was to utilize actual components to ensure it can be easily adapted for real-world applications.
